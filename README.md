# Windows-Activator
一个简易的 Windows 11 Professional 激活工具。
适用于 Windows 10-11 的可执行应用程序。

这是一个小工具，能够使用`slmgr.vbs`激活你的 Windows 11 系统。

### 是什么
它是一个基于 C++ WinAPI 的 Windows 激活工具，能够修改序列号并激活系统。
依赖于 WMI 模块和`slmgr.vbs`文件。
它使用控制台窗口主机界面，支持错误处理和着色显示。

### 为什么
在未激活的 Windows 11 操作系统上，你可以通过这个工具来激活 Windows 系统以享受更多体验。
使用 Windows 11 专业版密钥激活：
`W269N-WFGWX-YVC9B-4J6C9-T83GX`

### 怎么做
你可以在 Windows 桌面以管理员身份运行它来激活 Windows。
